# 未授权配置回退设计

## 边界

删除本轮自行新增的目标读者、叙事视角、风格要求、禁忌和发布节奏输入框、前端状态及必填校验；同时删除项目列表中同批新增的题材/类型及上述配置表单。不得回退用户明确要求的每章 3200-4000 字、卷章按内容与节奏动态规划、失败不降级、已有配置必须执行。

## 后端行为

创建和正文生成不得因为目标读者或叙事视角未填写而停止。若旧项目或其它合法流程已有这些配置，生成提示仍可读取；若没有，则保持前文、大纲或已确认素材中的叙事选择，不注入 `undefined`，也不擅自新增默认配置。

## 审计与验证

搜索本轮所有新增表单字段、必填校验、固定默认值和流程阻断。只移除未经授权的配置扩展，不整体回退其它已完成修复。完成两端类型检查、相关单测和生产构建。
