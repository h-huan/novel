# 当前系统 vs 参考文档 结构对比分析

## 三份参考文档体系

| 文档 | 角色 | 核心 |
|------|------|------|
| `两百万字小说创作全流程指南.docx` | **长篇总纲** | 8卷×50章, 400章, 200万字 |
| `短故事三步骤.md` | **短篇实现** | 3阶段: 题材→大纲(7章+反转表)→天龙8步正文 |
| `魂穿北洋，领众破局_修正版大纲.docx` | **长篇实例** | 第一卷106章, 每章详细内容, 时间线严格 |

**核心关系**: 短故事三步骤 = 把两百万字流程中的"基础设定+世界观+人物+大纲+伏笔"压缩为短篇可用版本

## 三、大纲结构

### 长篇 (200万字指南)
```
8卷(起承转合), 每卷25万字/50章
  卷1(1-50章): 引入+成长 → 主题/目标/关键事件(10-15)/卷末高潮
  卷2(51-100章): 扩展+危机
  ...
  卷8(351-400章): 收束+结局
  
  每章详细规划:
    - 章节标题
    - 核心内容(100字)
    - 主要场景(2-3个)
    - 人物行动
    - 冲突设计
    - 爽点设置
    - 伏笔设置
    - 伏笔回收
    - 下章钩子
```

### 短篇 (三步骤)
```
7章 + 开篇 + 尾声:
  开篇钩子(300字强异常)
  第1章: 异常降临
  第2章: 试探与误判  
  第3章: 危机升级
  第4章: 第一次大反转
  第5章: 真相逼近
  第6章: 高潮对峙
  第7章: 终局反转
  尾声: 余味结尾

  每章: 冲突+伏笔+钩子+反转点+情绪基调
```

### 魂穿北洋实例
```
第一卷 106章: "魂穿北洋，力挽狂澜止称帝"
  时间线: 1915.9.16 → 1916.6.6
  具体日期对应具体章节
  每章有核心内容和目标
```

| 维度 | 文档要求 | 当前实现 | 差距 |
|------|---------|---------|------|
| 长篇卷数 | 8卷 | 平铺无卷 | **需8卷层级** |
| 长篇章/卷 | 50章 | 6章 | **需扩展** |
| 短篇章数 | 7+2 | 6章 | 接近 |
| 章节内容 | 核心+场景+5+维度 | 仅有标题+content | **需扩展字段** |
| 时间线 | ✅ 日期精确 | ❌ | **缺** |
| 卷主题 | ✅ 每卷有 | ❌ | **缺** |

## 十、短篇创作专项要求

### 10.1 开篇钩子 (300字规则)

| 维度 | 三步骤要求 | 当前实现 | 差距 |
|------|----------|---------|------|
| 前300字 | ✅ 必须出现强异常 | ❌ 未约束 | **需在prompt中强制** |
| 读者疑问 | ✅ 产生必须继续看的问题 | ❌ | **缺** |
| 当时以为的真相 | ✅ 主角初始认知 | ❌ | **缺** |
| 实际隐藏真相 | ✅ 读者不知道但作者知道 | ❌ | **缺** |

**需要**: 短篇大纲生成prompt中增加"开篇300字钩子设计"

### 10.2 前三章吸引力

| 维度 | 三步骤要求 | 当前实现 | 差距 |
|------|----------|---------|------|

## 十一、长篇章节数量弹性规则

| 维度 | 魂穿北洋实例 | 当前实现 | 差距 |
|------|------------|---------|------|
| 卷数 | 根据剧情发展，非固定8卷 | 固定1卷或预设卷 | **需弹性** |
| 每卷章数 | 第一卷106章(远超50章预设) | 固定6章 | **需弹性** |
| 章节生成逻辑 | 尊重故事发展，不硬凑章节 | 固定生成 | **需LLM自主决定** |
| 卷主题 | 每卷有明确主题(如"力挽狂澜止称帝") | ❌ 无 | **缺** |

**核心原则**: LLM prompt 中不应固定"8卷×50章"，改为"根据故事自然发展阶段分卷，每卷5-150章不等，由剧情需要决定"。卷和章的数量由 LLM 根据题材和故事发展自主确定。

**需要修改 prompty**: 
- 删除 "分卷不超过8卷" 限制
- 删除 "每章 targetWords 3000" 固定值
- 改为 "根据故事节奏自然分卷，每卷章数据剧情需要(5-150章)"
- 增加 "每卷有明确主题和核心目标"
